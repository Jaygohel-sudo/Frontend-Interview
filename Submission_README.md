## Blog Platform (Frontend Interview Project)

A responsive blog platform built with React, TypeScript and Tailwind.

### Features

- Blog list with active state synced to URL
- Responsive collapsible sidebar for mobile
- Blog detail view with nested routing
- Create blog flow with POST request
- Loading skeletons and smooth transitions

### Tech Stack

- React + TypeScript
- React Router
- TanStack Query
- Tailwind CSS + shadcn/ui
- json-server (mock API)

### How to run

1. npm install
2. npm run server
3. npm run dev
